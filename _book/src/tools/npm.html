升级node与npm

一、Mac 升级node与npm
1、清除node.js的cache
sudo npm cache clean -f
2、安装 n 工具
sudo npm install -g n
3、安装最新版本的node.js
sudo n stable
sudo n vxxx
4、更新npm到最新版
sudo npm install npm@latest -g
5、更新npm到指定版
sudo npm install npm@xxx -g

二、window 升级node与npm

在https://github.com/coreybutler/nvm-windows/releases下载相应setup
